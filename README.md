Scripts and other stuff for IBM BigInsights
========================================

do.sh
-----
> **How to use it**:
> Edit the do.sh script in an editor. Change the nodes list for your node list. Change the permissions of the script, and run it!

```
# chmod +x ./do
# ./do “hostname”
```
